Chương trình APPi16 có gì nỗi bật

  • Cập Nhật những kiến thức mới về công nghệ Java và Microsoft.Net
  • Phát triển mô hình điện toán đám mây như : Saas, Iaas, Paas, Cloud apption in Sales
  • Phát triển ứng dụng với Microsoft và Google Appengenie
  • Phân tích, thiết kế và lập trình với Microsoft SQL sever
  • Xây dựng ứng dụng Web thế hệ mới với HTML, CSS3, BOOKTRAP và Javascript
  • Phát triển nền tảng di động Andriod Và Windows Phone
  • Học viên có đầy đủ kỹ năng làm việc nhóm và sản xuất sản phẩm ngay khi học xong HK I
  • Đồ Án cuối mỗi học kỳ(eprojet) làm việc trực với Ấn Độ
